HTML Cheatsheet

Command Description Use(s)
Table Commands: <table> This element contains organized data in table format. Used mainly for displaying table like things, but can also be used as a way of organizing, and stylizing, text.
<tbody> Identifies the body of the table. Although optional, tbody is used to depict the main body portion of the table.
<tr> Aka, the table row element. Starts a row for the table. In here, we will insert things like th or td's to order our data.
<th> The table header. Will store things that belong on the very top of a table to identify key elements down a column. This will bold whatever we put into it, its a header but in table form.
<td> This means table data. Table data is exactly what it says it is, cells of data within the table. Along with <th>, <td> is used within table rows. Table data is like paragraph text.
